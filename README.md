# Планировщик квартиры — ИИ-анализ планировок

Загружаете фото планировки → Claude AI анализирует → получаете PDF с 5–7 вариантами перепланировки на русском языке.

---

## Структура проекта

```
floorplan-app/
├── index.html     — фронтенд (сайт)
├── server.js      — бэкенд (Node.js сервер)
├── package.json
└── .env.example   — шаблон переменных окружения
```

---

## Локальный запуск (тестирование)

### Шаг 1 — Получите API-ключ Anthropic
1. Зайдите на https://console.anthropic.com
2. Раздел **API Keys** → **Create Key**
3. Скопируйте ключ (начинается с `sk-ant-...`)
4. Пополните баланс (минимум $5). Один анализ стоит ~$0.02

### Шаг 2 — Запустите бэкенд
```bash
# Установите зависимости
npm install

# Создайте файл .env
cp .env.example .env

# Вставьте ваш ключ в .env:
# ANTHROPIC_API_KEY=sk-ant-ваш-ключ

# Запустите сервер
npm start
```
Сервер запустится на http://localhost:3000

### Шаг 3 — Откройте фронтенд
Просто откройте `index.html` в браузере. Всё работает.

## Деплой (чтобы работало у всех пользователей)

### Бэкенд → Railway (бесплатно / $5 в месяц)

1. Зайдите на https://railway.app и войдите через GitHub
2. **New Project → Deploy from GitHub repo**
3. Загрузите файлы `server.js`, `package.json` в репозиторий GitHub
4. В Railway: **Variables** → добавьте:
   - `ANTHROPIC_API_KEY` = ваш ключ
   - `ALLOWED_ORIGIN` = URL вашего фронтенда (после деплоя)
5. Railway даст вам URL вида: `https://your-app.up.railway.app`

### Фронтенд → Vercel (бесплатно)

1. Зайдите на https://vercel.com и войдите через GitHub
2. Загрузите `index.html` в репозиторий GitHub
3. **New Project → Import** → выберите репозиторий
4. В `index.html` замените строку:
   ```javascript
   const BACKEND_URL = 'http://localhost:3000';
   ```
   на:
   ```javascript
   const BACKEND_URL = 'https://your-app.up.railway.app';
   ```
5. Vercel даст вам URL вида: `https://your-app.vercel.app`

### Финальный шаг
В Railway вернитесь в **Variables** и обновите:
```
ALLOWED_ORIGIN = https://your-app.vercel.app
```

Готово! Сайт работает для всех пользователей.

## Что входит в PDF

- **Обложка** — анализ исходной планировки, площадь, помещения
- **5–7 страниц** — каждый вариант: название, стиль, цветовая палитра, ключевые изменения, зонирование с площадями, список мебели, преимущества
- **Финальная страница** — сравнительная таблица всех вариантов + рекомендации
